onKeyDown

(
  • event
)

Defined in js/input.js:77

Event handler for document.onkeydown. Triggered when a key is pressed. If an action is bound to the key pressed, false is returned to cancel the default browser behavior of the keypress. If an action is not bound to the key pressed, true is returned allowing the default browser behavior of the keypress to function.

Parameters:
